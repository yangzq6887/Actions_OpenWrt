# 云编译 OpenWrt 固件

**项目说明**：
- 本项目使用 Github Actions 下载 [Lean](https://github.com/coolsnowwolf/lede) 的 `Openwrt` 源码仓库，进行云编译。
- 本项目使用定时编译（北京时间每周日下午4点开始自动运行编译）及触发编译（更新script.sh后可开始编译）两种方式。
- 本项目编译固件适配 X86_64架构
- 本项目编译固件后台地址：192.168.1.2 管理员：root  初始密码：空
